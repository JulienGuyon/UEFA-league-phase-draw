# Simulation Results

This repository contains the results of **40,000 simulated draws** for UEFA league-phase competitions.

Each results file corresponds to a specific draw algorithm and competition.  
Each file contains exactly 40,000 lines, where:
- Each line represents one simulated draw
- Each line enumerates all pairs of opponents drawn in that simulation

The following acronyms are used throughout the repository:
- **bsf** — *Build Schedule First*
- **dmf** — *Draw Matchups First*
- **rd** — *Randomized variant*
- **ucl** — *UEFA Champions League (2024–2025)*
- **uel** — *UEFA Europa League (2024–2025)*

A typical filename encodes:
- the competition,
- the draw algorithm,
- and whether a randomized variant is used.
